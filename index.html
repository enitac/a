Portal User Manual

1. Secondary Portal (Front Page)

The front page is designed for students, parents, guardians and sponsors to access exam results quickly.

How to View Results:

  • On the home page, click Get Started button.
  • Choose your portal: Exam Day or Exam ODL.
  • Select Form: Choose the student's class (Form 1-4).
  • Enter Exam Number: Enter the unique ID (e.g., J047 or 004).
  • Enter Password: (Default unless disabled by Admin).
  • Click View Results for the official Progress Report.

First-Time Token Setup

Encrypt your GitHub Personal Access Token with a password. The encrypted token will be embedded in this HTML file.

Instructions:
  1. Enter your GitHub token (starts with ghp_...)
  2. Choose a strong password (this is your login password)
  3. Click "Encrypt Token"
  4. Click "Copy to Clipboard"
  5. Open this HTML file in a text editor
  6. Find: __ENCRYPTED_TOKEN_HERE__
  7. Replace it with the copied encrypted text
  8. Save and re-upload to GitHub
